# Навигатор по современной медицине

Интерактивный информационно-образовательный сайт для учебного проекта по дисциплине «Основы российской государственности».

Проект показывает, как национальные проекты и государственные инициативы в сфере здравоохранения связаны с доступностью медицинской помощи, профилактикой, инновациями, экстренной помощью и цифровыми технологиями в России.

## Стек

- React
- Vite
- CSS без backend и CMS
- Hash-навигация для корректной работы на GitHub Pages

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Предпросмотр сборки

```bash
npm run preview
```

## Деплой на GitHub Pages

В `vite.config.js` задан base:

```js
base: process.env.NODE_ENV === 'production' ? '/org-project/' : '/'
```

Если репозиторий на GitHub будет называться иначе, замените `/org-project/` на `/<repo-name>/`.

После этого:

```bash
npm run build
npm run deploy
```

Скрипт `deploy` использует пакет `gh-pages` и публикует папку `dist`.

## Структура сайта

- Главная
- О проекте
- Разделы сайта
- Борьба с онкозаболеваниями
- Развитие НМИЦ и инновационных медицинских технологий
- Борьба с сердечно-сосудистыми заболеваниями
- Совершенствование экстренной медицинской помощи
- Источники

## Структура кода

- `src/components` - переиспользуемые UI-блоки
- `src/data` - контент, источники, команда, показатели
- `src/pages` - страницы сайта
- `src/index.css` - общая визуальная система и адаптив

## Картинки разделов

Картинки нужно сложить в папку `public/section-images`.

Ожидаемые имена файлов:

- `1.png` - онкология
- `2.png` - НМИЦ и инновационная медицина
- `3.png` - сердечно-сосудистые заболевания
- `4.png` - экстренная медицинская помощь

Если файла пока нет, сайт покажет аккуратную заглушку. Если используете JPG или WebP, измените поле `image.file` в `src/data/sections.js`.

## Файлы для скачивания

Файлы материалов лежат в `public/files` и подключены к страницам разделов кнопками скачивания:

- `onkologiya_proekt_materialy (1).docx` - раздел онкологии
- `материалы проект.docx` - раздел НМИЦ и инновационной медицины
- `отчёт.pdf` - раздел сердечно-сосудистых заболеваний
- `Совершенствованние_экстренной_медицинской_помощи.docx` - раздел экстренной помощи
